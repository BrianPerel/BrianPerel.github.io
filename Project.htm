Company Employee Management App



For this project, I developed a GUI employee management application.

- Program has add, update, delete, look-up, load and reset actions
- Used a dictionary whose keys store the employee's ID and values store the employees attributes
- Used an employee class that sets and returns attributes
- Used mysql connector and xampp to connect the program to mysql and store data in a database
