ATM Machine App

For this project, I developed a Java ATM simulation app.

- ATM functions: select savings or checkings, balance inquiry, deposit, withdraw, transfer funds,
terminate account, save and load object
- Withdraw operation will be prevented if balance is below withdraw amount
- Deposit operation will be prevented if deposit amount is extremely high
- Save information to receipt txt file, prompt user if receipt is wanted, if not then receipt file is deleted
