ftp Simulator



For this project, I developed a client-server netcentric program in C that uses the ftp and tcp protocols.

- The client program connects to the server side program using socket
- User must enter a valid ftp username and password pair
- User enters cmd and argument, cmd is sent to server
- Server program checks cmd and argument, performs appropriate implementation and returns message to client
- Client program prints reply message to user
- Example cmds include: stat, help, ls, cd, pwd, rm, send, mkdir, rmdir, recv, quit

Link to project on Github: https://github.com/BrianPerel/client-server-ftp
